### <a name="inventory-server-capacity"></a>Utworzenie spisu pojemności serwerów

Zapisz składniki sprzętowe (pamięć, procesor, dysk) bieżących serwerów produkcyjnych, a także średnią i szczytową liczbę żądań oraz użycie zasobów. Te informacje są ważne w kontekście wyboru rozmiaru maszyny wirtualnej. Aby uzyskać więcej informacji, zobacz [Rozmiary usług Cloud Services](/azure/cloud-services/cloud-services-sizes-specs).
